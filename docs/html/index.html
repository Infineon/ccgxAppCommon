CCGx App Common
CCGxAppCommon Middleware Application 1.0

Overview

The CCGxAppCommon Middleware application contains the top-level application layer functionality that implements the required WLC1 controller functions.

The CCGxAppCommon Middleware is released in the source form for use with WLC1 sink based applications.

Features

1) Support following features:

  • PDO evaluation
  • Contract negotiation
  • Legacy charger detection
  • Type-A port controller detection
